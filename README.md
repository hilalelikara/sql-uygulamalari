# sql-uygulamalari
Bu depo, temel ve orta seviye SQL sorgularını içeren örnek uygulamalardan oluşmaktadır.  

# Hedeflenen Kazanımlar

- SQL sorgu yapısına hakimiyet  
- Veri seçme, filtreleme ve sıralama işlemleri  
- Birden fazla tablo ile ilişkilendirme (JOIN işlemleri)  
- Veritabanı üzerinde işlem otomasyonu (Trigger ve View)

Bu uygulamalar öğrenme ve portföy geliştirme amacıyla hazırlanmıştır.
